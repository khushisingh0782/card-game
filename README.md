# Image Card Gallery

A simple and visually appealing image card gallery built using HTML and CSS.

## 📌 Features

* Clean card-based layout
* Four image cards with high-quality photographs
* Responsive structure
* Easy to customize and expand
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```
project-folder/
│
├── index.html
├── style.cards.css
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## 🎨 Customization

You can:

* Replace the image URLs with your own images.
* Change card sizes and spacing in `style.cards.css`.
* Add animations and hover effects.
* Make the gallery fully responsive for mobile devices.

## 📸 Preview

The project displays four image cards arranged in a stylish layout using HTML and CSS.
